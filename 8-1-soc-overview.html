SOC Key Performance Indicators

KPI Target Measurement
Mean Time to Detect (MTTD) < 15 minutes Time from attack start to alert generation
Mean Time to Respond (MTTR) < 1 hour (critical) Time from alert to containment action
Alert Closure Rate > 95% Alerts resolved within SLA
False Positive Rate < 10% Benign alerts / total alerts
Secure Score > 80% MDC secure score percentage
Coverage 100% Resources with Defender enabled

KPI Dashboard Query

KQL 
// SOC KPIs - Alert metrics last 30 days
SecurityAlert
| where TimeGenerated > ago(30d)
| summarize 
    TotalAlerts = count(),
    HighSeverity = countif(AlertSeverity == "High"),
    CriticalSeverity = countif(AlertSeverity == "Critical"),
    Resolved = countif(Status == "Resolved"),
    AvgTimeToResolve = avg(datetime_diff('minute', 
        todatetime(ExtendedProperties.ResolvedTime), TimeGenerated))
| extend 
    ResolutionRate = round(100.0 * Resolved / TotalAlerts, 1),
    CriticalRate = round(100.0 * CriticalSeverity / TotalAlerts, 1)

Automation Opportunities

Alert Enrichment
  • Add threat intelligence context (IP reputation, domain info)
  • Lookup asset ownership from CMDB
  • Attach recent user activity
  • Include vulnerability scan results
See Enrichment Playbooks â†’
Auto-Remediation
  • Isolate VM on critical malware detection
  • Block storage account on malware upload
  • Revoke user sessions on credential theft
  • Apply NSG deny rules on C2 communication
Caution

Auto-remediation can cause service disruption. Start with notification-only, then enable for well-understood scenarios.

Notification & Ticketing
  • Create ServiceNow/Jira tickets for alerts
  • Send Teams/Slack notifications
  • Email alert summaries to management
  • PagerDuty integration for on-call

Best Practices

SOC Operations Checklist
Key Takeaways
  • Use tiered SOC model: Tier 1 monitoring, Tier 2 investigation, Tier 3 IR
  • Assign appropriate RBAC roles per tier (Reader vs Admin)
  • Create playbooks for each major alert category
  • Automate enrichment, notification, and simple remediations
  • Track KPIs: MTTD, MTTR, closure rate, false positive rate
  • Integrate MDC with Sentinel for unified SOC operations
  • Regular training and tabletop exercises improve response
  • Document everything for compliance and improvement
